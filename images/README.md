# /images

Folder ini untuk menyimpan foto banner artis konser.

## Cara penggunaan

1. Upload foto artis ke folder ini (JPG/PNG/WebP, maks 1MB, resolusi min 600x400px)
2. Nama file gunakan format: `[concert-id].jpg`
   - Contoh: `bts-jakarta-2026.jpg`, `westlife-jakarta-2027.jpg`
3. Update URL di `app.js` → `ARTIST_IMAGES`:
   ```js
   'bts-jakarta-2026': '/images/bts-jakarta-2026.jpg',
   ```

## Daftar artis yang perlu foto

| Concert ID | Artis | Status |
|---|---|---|

| `perses-jakarta-2026` | Perses | ❌ Belum ada |
| `fforever-jakarta-2026` | F✦FOREVER | ❌ Belum ada |

## Sumber foto yang direkomendasikan

- Foto official dari akun Instagram/Twitter resmi artis
- Screenshot dari YouTube MV/concert official
- Foto dari press release promotor

> ⚠️ Pastikan foto yang diupload bukan untuk keperluan komersial.
